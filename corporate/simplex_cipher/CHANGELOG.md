# Changelog - SimplexCipher

## [3.0.0] - 2025-12-20 - 🟢 Production Era

### Added
- Política de cifrado con mínimos de keysize y AEAD por defecto.
- Alertas para ciphers legacy (DES/RC4/3DES/AES-128).
- Generación de llaves seguras (base64 urlsafe).
- Cifrado simulado con HMAC-SHA256 para integridad.
- Modelos Pydantic y tests de roundtrip y política.
